# 🍅 番茄钟

一个简洁好用的番茄工作法计时器，支持自由设置专注时长。

## ✨ 功能

- 🕐 自由设置倒计时分钟数
- ⏯️ 开始 / 暂停 / 停止控制
- 🎨 圆形进度条 + 颜色渐变（绿 → 黄 → 红）
- 💫 每秒数字弹跳动画
- ✨ 完成时闪烁提示

## 🖥️ 使用方式

### 网页版（在线使用）
直接访问：**[hhhhkkkkd.github.io/tomato-timer](https://hhhhkkkkd.github.io/tomato-timer)**

### Mac App（原生）
1. 下载 Xcode 项目
2. 用 Xcode 打开，运行即可

## 🛠️ 技术栈

- **macOS App**: SwiftUI + Combine
- **网页版**: HTML + CSS + JavaScript
