# 🍅 复习计时器 · Revision Timer v1.1

像素复古风番茄钟，专为专注学习设计。

## ✨ 功能

- 🕐 自由设置倒计时分钟数
- ⏯️ 开始 / 暂停 / 停止控制
- 🟢 像素风虚线进度环（绿 → 黄 → 红渐变）
- 💫 每秒数字心跳弹跳动画
- 🖨️ 倒计时结束自动吐出复古收据
- 🖥️ 经典 CRT 终端黑底绿字风格

## 🚀 使用方式

### 🌐 网页版
**[hhhhkkkkd.github.io/tomato-timer](https://hhhhkkkkd.github.io/tomato-timer)**

### 💻 Mac App
1. `git clone https://github.com/hhhhkkkkd/tomato-timer.git`
2. Xcode 打开项目，运行即可
