# 🍅 REVISION TIMER v1.1

> 像素复古风番茄钟 | 黑底绿字 CRT 终端美学

---

## ✨ 功能

- 🕐 自由设置倒计时分钟数
- ⏯️ 开始 / 暂停 / 停止控制
- 🟢 像素虚线进度环（绿→黄→红）
- 💫 每秒数字心跳弹跳动画
- 🖨️ 倒计时结束弹出复古收据
- 🎨 全黑背景 + 荧光绿字

---

## 🚀 使用方式

### 🌐 网页版
**[hhhhkkkkd.github.io/tomato-timer](https://hhhhkkkkd.github.io/tomato-timer)**

### 💻 Mac App
```bash
git clone https://github.com/hhhhkkkkd/tomato-timer.git
```
Xcode 打开 `复习计时器.xcodeproj` 运行即可。
